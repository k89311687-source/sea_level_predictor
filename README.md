# Sea Level Predictor

## Description
This project analyzes global sea level changes and predicts future rise using linear regression.

## Features
- Scatter plot visualization
- Linear regression modeling
- Future prediction till 2050

## Technologies Used
- Python
- Pandas
- Matplotlib
- SciPy

## How to Run
1. Install dependencies
2. Run: python main.py

## Output
- sea_level_plot.png
